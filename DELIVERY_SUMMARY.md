```
Neutral Light:     #ecf0f1 (Clean background)
Text Dark:         #1a1a1a (Primary text)
Text Light:        #555555 (Secondary text)
White:            #ffffff (Main background)
```

### Typography
```
H1: 3.5rem (Hero headlines)
H2: 2.5rem (Section titles)
H3: 1.8rem (Subsection titles)
H4: 1.4rem (Card titles)
H5: 1.1rem (Small titles)
Body: 1rem (16px standard)
Small: 0.875rem (14px)
Font Family: Segoe UI, sans-serif
```

### Spacing System
```
Base Unit: 1rem (16px)
Options: 0.5rem, 1rem, 1.5rem, 2rem
Applied to: Margins, padding, gaps
Mobile Adjustments: 20px default padding, reduced gaps
Tablet Adjustments: 15px padding
Responsive padding on sections: 80px > 60px > 40px
```
